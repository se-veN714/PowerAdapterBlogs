关于本站

这是一个以技术、写作与生活反思为主题的个人博客，采用 Django + Bulma 构建。 内容涵盖编程、开发日志、思想片段、音乐随笔等。欢迎你在这里探索与共鸣。

Skateboard｜滑板

街头、风格与坚持。
这一栏是我记录滑板学习历程的地方，从基础动作练习到刷街心得，也会分享我喜欢的滑手、视频剪辑，以及那些让我心跳加速的滑行片段。 欢迎跟我一起享受沾满灰尘的自由。

Music｜音乐

这里收藏我喜欢的声音。
从 hardcore 到 post rock，从 Rough Sketch 到 world's end girlfriend，不管是解构节奏的 math rock，还是带有诗意的噪音。偶尔也会分享一些吉他练习、乐理学习笔记或翻奏尝试。

Coding

不止代码，更是对技术结构的好奇。
这里会放上我学习过程中的笔记、 实践与反思——可能是后端开发，也可能是前端样式、AI 理论，甚至是某次命令行里的小发现。风格不拘一格，但始终追求优雅与效率。
